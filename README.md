# Projektin työaikaseuranta

## Esittely

Ohjelmalla voi seurata useampien projektien työaikoja henkilöittäin. Peruskäyttäjä näkee vain omat kirjauksensa, projektipäällikkö saa raportit kaikista projektin tunneista.
